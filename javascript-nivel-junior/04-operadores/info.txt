Operadores en Javascript

- Operadores de asignacion
    # Asigna un valor al operando de la izquierda basado en el valor del operando de la derecha.

    x = y
    x += y
    x -= y
    x *= y
    x /= y
    x %= y
    x **= y => Asignacion de exponienciacino
    x <<= y => Asignacion de desplazamiento a la izquierda
    x >>= y => Asignacion de desplazamiento a la derecha
    x >>>= y => Asignacion sin signo de desplazamiento a la derecha
    x &= y => Asignacion AND
    x ^= y => Asignacion XOR
    x |= y  => Asignacion OR
